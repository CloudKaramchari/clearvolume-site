What ClearVolume Collects

ClearVolume does not collect, transmit, or sell any personal information, browsing history, or search queries to the developer or any third party.

The current version stores the following locally on your device, using Chrome's built-in storage APIs:

  • Your daily keyword-lookup count (resets each day), stored via chrome.storage.local.
  • Your site-toggle preferences and display settings, stored via chrome.storage.sync so they follow your Chrome profile across devices signed into the same Google account. This data is only ever visible to you.

What ClearVolume Does Not Do

  • No browsing history is read, recorded, or transmitted.
  • No analytics or third-party tracking scripts are included.
  • No advertising network is integrated.
  • No data is sold or shared with third parties.

Permissions Explanation

storage — used only for the local counters and settings described above.

Host permission for google.com — used only to detect the search query on Google's results page (read from the page URL, not page content) so the inline data card can render. No other site content is read or transmitted.

Future Changes

If a future version adds an account system or server-side keyword-data lookups, this policy will be updated to disclose exactly what is sent to ClearVolume's servers and how long it is retained, before that version is published.
